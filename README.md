# Social Media Platform

A full-stack social media application leveraging modern technologies for real-time interactions and intelligent features.

## Project Structure

- **`ml/`** - Python machine learning models and services
- **`backend/`** - Jakarta EE 10 backend with WebSocket support and, Data management and processing
- **`frontend/`** - React.js user interface
- **`infra/`** - Infrastructure and deployment configuration

## Tech Stack

- **Backend**: Jakarta EE 10, WebSocket
- **Frontend**: React.js
- **ML**: Python
- **Infrastructure**: Docker, Kubernetes, kafka,  redis
